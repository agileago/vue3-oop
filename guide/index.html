如何使用 #

前提条件 #

  1. 因为要用到ts的获取元数据的能力，所以需要安装reflect-metada的支持
yarn add @abraham/reflection

并且把这段代码放到入口引入，只需引入一次

import '@abraham/reflection'
  1. 安装依赖注入库 injection-js, 以及本库
yarn add injection-js vue3-oop
  1. 设置 tsconfig.json

主要是涉及到装饰器以及类的设置

{
  "compilerOptions": {
    "experimentalDecorators": true,
    "emitDecoratorMetadata": true,
    "useDefineForClassFields": false,
  }
}
  1. 如果使用 vite 的话，由于vite使用esbuild编译ts,不支持 metadata , 需要把 @vitejs/plugin-vue-jsx 这个插件换成vite-plugin-ts这个插件
yarn add vite-plugin-ts

vite 插件配置

import vueJsx from 'vite-plugin-ts'
export default {
  plugins: [vueJsx()]
}
